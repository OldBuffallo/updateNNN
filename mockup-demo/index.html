💬 Câu hỏi thường gặp

Tôi quên mật khẩu?

Liên hệ quản trị viên để đặt lại mật khẩu. Vào Quản trị → Tài khoản → Đặt lại mật khẩu.

File Excel import bị lỗi?

Đảm bảo file đúng định dạng .xlsx. Dòng đầu tiên phải là tiêu đề cột. Dữ liệu bắt đầu từ dòng 2.

Làm sao xuất báo cáo?

Vào Báo cáo → Chọn cột & điều kiện → Nhấn "Xuất Excel" hoặc "Xuất PDF". Có thể lưu template để dùng lại.

NLĐ sắp hết hạn hiển thị ở đâu?

Trên Dashboard có bảng cảnh báo. Hoặc vào Báo cáo → chạy template "BC NLĐ sắp hết hạn tạm trú".
